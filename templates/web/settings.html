テクニカル 50点

  • MA5上抜け +10
  • 前日比反発 最大10（前日比% ÷ 反発率閾値 × 10）
  • 急落時から回復 最大15（急落時比% ÷ 回復閾値 × 15）
  • 出来高急増 最大10（(倍率−1) ÷ (閾値−1) × 10）
  • RSI回復 +5（直近でRSI30未満あり、現在35以上）

ファンダ 30点　赤字は0点固定

  • PER 最大10（<10倍=10 / <15=8 / <20=5 / それ以上=2 / 不明=5）
  • PBR 最大10（<1倍=10 / <1.5=8 / <2=5 / それ以上=2 / 不明=5）
  • 配当利回り 最大10（3%以上=10 / 2%=7 / 1%=4 / 1%未満=1 / 不明=3）

地合い 20点　急落日の日経下落ほど高得点

  • 日経 −2%以上=20 / −1%以上=14 / 0%以下=8 / 上昇日=4 / 不明=10
